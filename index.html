Intro

눈에 보이는 프론트단에 관심이 많지만 저의 경험과 잠재력을 넓히기위해 지난 8개월동안 대덕인재개발원에서 풀스택 과정을 수료하였습니다.

프론트앤드개발자로서 부족한 부분은 독학하여 flex그리드, 동적 애니메이션, 미디어쿼리를 통한 반응형 UI등을 최종프로젝트에 반영하였습니다.

우물안 개구리가 아닌 우물밖 개구리가 되어 항상 배우며 성장하고 싶습니다.

Middle Projects

Car Sharing System

Java, Javafx

고급 자바를 마치고 진행한 중급 프로젝트는 ‘CSS(Car Sharing System)’이라는 고급 차량 대여 서비스를 주제로 진행되었습니다. JavaFX를 활용하여 UI를 꾸미면서 구글링을 본격적으로 하게 되었습니다. 스택오버플로우나 유튜브를 검색하여 메뉴바 슬라이드와 팝업창, 자바 영상 구현을 완성하였습니다. 게시판 CRUD 기능을 해보는 경험을 가질 수 있었습니다.

2020.1.20 ~ 2020.2.13

JSP CRUD Project

Java, JSP, Bootstrap, Jquery

JSP 수업 중에 진행한 미니 프로젝트는 3명의 팀원이 각자 게시판을 맡아 구현하였고 회원가입과 회원 탈퇴 시에 회원가입, 탈퇴 사유를 전달하기 위해 디버깅을 활용하는 방법을 알게 되었습니다. 데이터베이스부터 VO, 컨트롤러, 각종 JSP 파일들을 직접 구현해보면서 부트스트랩을 사용하였고 회원 탈퇴 요청 이후로 로그인이 가능하지 않게 하는 기능을 구현해보았습니다.

2020.1.20 ~ 2020.2.13

Final Project

개미의 집

Java, Spring, Html, CSS, JavaScript

UA로써 부트스트랩 없이 HTML, CSS, Java Script로 웹 애플리케이션을 만들고 싶은 욕심이 생겼고 기다리는 팀원들이 편의를 생각해 혼자서 한달안에 메인 디자인과 모든 UI 페이지들을 빨리 마무리해야 된다는 스트레스와 압박감이 상당하였습니다. 이점을 팀원들에게 솔직하게 터놓게 되었고 혼자 하려 했던 부담을 내려놓아 몇 개의 UI가 간단한 페이지는 팀원들에게 분담하고 함께 브랜딩을 정하되 UI가 계속 바뀌는 상황이기 때문에 메인을 마칠 때까지만 CSS를 UA만 건드는 것으로 양해를 구하였습니다. 이로써 더욱더 서로를 믿고 돈독해지는 계기가 되었고 커뮤니케이션의 중요성을 깨달았습니다.

2020.1.20 ~ 2020.2.13
